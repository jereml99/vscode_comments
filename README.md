# Git Review Comments

Inline review comments stored in the repo.

## Features

- Add review comments directly in your code.
- Comments are stored in `.review-comments.json` in your workspace.
- Share comments via Git.
- Anchor comments to code context.

## Usage

1. Select code to comment on.
2. Use `Ctrl+Alt+C` (or `Cmd+Alt+C` on Mac) to add a comment.
3. View threads in the Review Comments sidebar.

## Requirements

VS Code ^1.90.0
